# NodeJS 프로젝트 2022

1. NodeJS 프로젝트 폴더 만들기 : `mkdir project명`
2. 프로젝트 폴더에 진입 : `cd project명`
3. NodeJS 프로젝트 생성 : `npm init -y`
4. 부모 폴더로 이동 : `cd ..`
5. git hub에 push하기 : `git add .`, `git commit`, `git push`
6. 왼쪽 프로젝트 목록에서 프로젝트 폴더 우클릭, 터미널 열기
